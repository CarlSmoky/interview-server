### Instration
1. Clone this repository
```console
$ git clone git@github.com:CarlSmoky/interview_server.git
```

2. Install packages
```console
$ cd interview_server
$ npm install
```

3. Start
```console
$ npm run start
```


For running this app's client, go to [client repo](https://github.com/CarlSmoky/interview-client)
